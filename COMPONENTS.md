## Animation Library

All animations use **Framer Motion**:

```tsx
// Fade in on scroll
<motion.div
  initial={{ opacity: 0, y: 50 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  transition={{ duration: 0.6 }}
/>

// Hover scale
<motion.button
  whileHover={{ scale: 1.05 }}
  whileTap={{ scale: 0.95 }}
/>

// Infinite animation
<motion.div
  animate={{ y: [0, -20, 0] }}
  transition={{ 
    duration: 4, 
    repeat: Infinity 
  }}
/>
```
